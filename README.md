## Setup
#### Clone this repository
```bash
git clone https://github.com/SaranshDawra/Pixel-Pit.git
```
#### Move to desired directory
```bash
cd Pixel-Pit
```
#### To install the dependencies
```bash
npm install
```
#### To run the app
```bash
npm start
```
